Encrypt and Decrypt files using 7-zip and Batch Scripting

Ever want a quick and easy way to encrypt and then decrypt single files on your computer? Well you're in luck because while bored, I found an interesting way to do this using 7-zip and a batch script. Why not Powershell? I was simply feeling a little nostalgic and decided to write out a shell script the old fashioned way. This one is a relatively simple task.

  1. First thing you have to make sure you do is go out and download the latest installer for 7-zip. Choose your preferred installer here. Install it like you would any other software.
  2. Time to write a little script. Fire up your favorite text editor and create a file called "encrypt.bat".
  3. Enter the following code:
    @echo off
    "C:\Program Files\7-Zip\7z.exe" a %1.7z %1 -p
    del %1
    Save this file when completed.
  4. Create a new file and call it "decrypt.bat". Enter the following code below:
    @echo off
    "C:\Program Files\7-Zip\7z.exe" e %1
    Save this file when completed.
  5. That's it! You're ready to put this to use.

You can now utilize these scripts via command line or windows explorer by just dragging and dropping a file onto the batch script. The encrypt script works by adding your file to an encrypting archive that you then apply a password to. Take note of that password for when you decrypt, you will take the .7z file and run it through the decrypt.bat. Take note that in the above code examples the "Program Files" path is used. This path may be different if you decide to use the x86 version ("Program Files (x86)"). Enjoy!
